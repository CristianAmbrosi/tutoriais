

					Instalar Conky Cronograph no Ubuntu


1º Antes de tudo, atualizar os repositórios e gatilhos do Ubuntu:

	$ sudo apt-get update

	$ sudo apt-get upgrade


2° Baixar o arquivo zipado neste link e descompacte:
	- https://github.com/drxspace/cronoconky


3º Executar dentro da pasta extraída o arquivo "install.sh" no terminal (clicar duas vezes como um arquivo .exe). Seguir os seguintes passos:

	- Do you want to continue? [Y/n]: Y

	- Do you want to continue? [Y/n]: Y

	- Do you want to install the hddtemp service as daemon? [Y/n]: n

	- Celsius or Fahrenheit? [C/f]: Opcional

	- Do you want to add this conky to the Startup Applications list? [Y/n]: Y

	- Do you want to start the conky right now? [Y/n]: Y


4º Acessar o site Yahoo Weather, alterar F (Fahrenheit) para C (Celsius) ou vice-versa e procurar pela cidade desejada. Ao selecionar a cidade, na URL do navegador copiar o código da cidade.

	- Exemplo: https://weather.yahoo.com/brazil/rio-grande-do-sul/erechim-457152/

	- https://weather.yahoo.com/


5º Ir na pasta Home do sitema, precionar Crtl+h para exibir as pastas ocultas. Entrar na pasta ".conky/cronograph_se/yahooweather" e abrir o arquivo "forecasts.sh". Na linha 174, alterar a variável "WOEID='946738'" pelo código obtido no site Yahoo Weather e salvar. Volte uma pasta e execute o arquivo "restart_crono.sh" restartar o programa para iniciar com as configurações feitas.


6º Se aparecer uma sombra inativa na janela do Conky Trusty Tahr Unity, abrir o arquivo na área de trabalho "decor-shadow" e utilizar esta solução para esconder:

  
  /*===================================================================
  * From Conky-Harmattan Troubleshooting
  * I GET A STRANGE SHADOW AROUND MY CONKY(I USE COMPIZ)!
  * =================================================================*/
		
		You are likely using compiz.
		Since "own_window" is set to "yes" compiz will also draw shadows on it like on any other window!
		To prevent compiz from drawing a shadow around conky open the compizconfig-settings-manager!
		If you don´t have it installed then install it via synaptic-package manager or terminal(sudo apt-get install compizconfig-settings-manager)
		Go to the "window-decorations".
		To exclude conky type "any& !(class=Conky)" without quotes in the window shadow entry!
		Conky´s class is by default set to "Conky".
		If, for whatever reason, it isn´t then just add "own_window_class Conky" to the conky file.
  /*=================================================================*/
		
# any & !(class=Conky)
# sudo apt-get install -y compizconfig-settings-manager dconf-editor gconf-editor
# gsettings list-relocatable-schemas | grep unityshell

gsettings set org.compiz.unityshell:/org/compiz/profiles/unity/plugins/unityshell/ override-decoration-theme true
gsettings set org.compiz.unityshell:/org/compiz/profiles/unity/plugins/unityshell/ inactive-shadow-color "#00000000"
